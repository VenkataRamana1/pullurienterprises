1. What is Java?

Answer:

Java is a high-level programing language and platform independent which is developed by Sun Micro Systems and later it bought by Oracle. Java is a general-purpose programming language that is class-based, object-oriented, and desinged to have as few implemenation dependencies as possible.

The platform consists of two elements:

  • Java Runtime Environment(JRE), which contains the parts of the Java SE platform needed to run Java applications.
  • Java Development Kit (JDK), includes development tools such as the Java compiler,Jar and a debugger, Javadoc.
